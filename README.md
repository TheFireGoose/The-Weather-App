# The-Weather-App
Project of weather app

In project of weather app you will receive information of current state of weather in city

1 Find your APi key from api.openweathermap.org
2 Copy and paste your APi key in test.js
3 Open test.html and start program
4 Write namy of any city in input you will receive information about location current temperatior,wind speed min and max temperatiour of the day
5 On the left of input located all of your favourite cities
